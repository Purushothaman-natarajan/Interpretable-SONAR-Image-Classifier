Clustering Features

Advanced unsupervised algorithms for underwater image analysis

K-Means Clustering

Partition SONAR images into K clusters based on pixel intensity and texture features for seabed segmentation.

DBSCAN

Density-based clustering for identifying anomalies and irregular structures in underwater SONAR imagery.

Hierarchical Clustering

Agglomerative and divisive clustering for discovering nested structures in SONAR image regions.

Feature Extraction

CNN-based feature embeddings for clustering, using pre-trained model representations as cluster inputs.

Visualization

PCA, t-SNE, and UMAP projections for visualizing high-dimensional clustering results.

Auto Parameter Tuning

Automatic selection of optimal cluster count using silhouette score and elbow method.
