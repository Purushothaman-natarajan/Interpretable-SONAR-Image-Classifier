Key Features

Advanced deep learning with explainability for underwater object detection

XAI Integration

Built-in support for LIME, SHAP, and Grad-CAM to provide transparent model explanations and interpretable predictions for critical underwater detection tasks.

Transfer Learning

Leverage pre-trained models (VGG16, ResNet50, DenseNet121, etc.) trained on ImageNet for robust underwater image classification.

SP-LIME Support

Submodular Picks LIME for selecting representative superpixels and generating comprehensive visual explanations of model decisions.

Image Clustering

Unsupervised learning for SONAR image segmentation using K-Means, DBSCAN, and hierarchical clustering. Learn more

Multiple Backbones

Support for 14+ CNN architectures including VGG16/19, ResNet50/101, InceptionV3, DenseNet121/201, MobileNetV2, and EfficientNet variants.

Multi-Dataset Support

Works with Seabed Objects KLSG, Camera SONAR, Mine SONAR, and SCTD datasets for comprehensive underwater object classification.

Performance Metrics

Comprehensive evaluation with accuracy, precision, recall, and F1-score for model performance assessment and comparison.

Quick Start

Get up and running in minutes

1. Setup Environment

Terminal 
conda env create -f environment.yaml
conda activate Interpretable-SONAR-Image-Classifier

2. Train a Model

Terminal 
python train.py --base_models "VGG16" "DenseNet121" \
  --shape 224 224 3 \
  --data_path "./processed_data" \
  --epochs 100 --optimizer "adam"

3. Generate Explanations

Terminal 
python classify_image_and_explain.py \
  --image_path "./images/test.jpg" \
  --model_path "./models/model.keras" \
  --explainer_types "lime,gradcam" \
  --output_folder "./explanations"
